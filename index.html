How to play

Use each of the given numbers exactly once, along with arithemetic operatorations to equal a target number.

Target 10

1+2+3+4 = 10

This would be a valid solution, but so would this:

14/2+3 = 10

Remember the only rule is to use each number once, those numbers do not have to be separated by an operator.

Most problems will have multiple solutions. There are in fact 7 unique ways to solve this example! Here are the other 5:

4*2+3-1
1*2*3+4
4*3-2/1
3*2+4/1
3*4-2*1
